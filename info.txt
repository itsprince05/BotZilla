=================================================
FILE DESCRIPTIONS
=================================================

1. CODE FILES
-------------
- bot.py          : Main Python script for the Pyrogram Telegram bot. Contains all command handlers, async download logic, DRM decryption workflow, queue system, and concurrency limit management.
- dashboard.py    : Flask backend script. Contains all API routes (/api/shows, /api/buyers, etc.) and web view routes to serve the UI dashboards.
- db.py           : Central database helper script. Contains all functions to read and write from JSON storage files safely.
- dashboard.html  : HTML/JS frontend template for the main dashboard (Add Shows, View Buyers, View All Users).
- user_shows.html : HTML/JS frontend template for specific buyer settings (Allowed Shows list, Total Shows toggle list, Setting tab for artist/cover toggles and names).
- show_users.html : HTML/JS frontend template to toggle which users have access to a specific Show.

2. DATABASE FILES (JSON)
------------------------
- shows.json         : Persistent database file storing all added shows, their IDs, and their MPD decryption keys.
- allowed_users.json : Persistent database file storing allowed buyers, their permissions, selected shows, and customization settings (cover/artist/status).
- all_users.json     : Persistent database file storing a list of everyone who has ever interacted with the bot.
- admins.json        : Persistent database file storing Telegram IDs of secondary admins allowed by the owner.
- pocketfm_auth.json : Persistent database file containing dynamic API tokens and auth details for PocketFM extraction.

3. ENVIRONMENT & DEPENDENCIES
-----------------------------
- .env            : Environment variable configuration file storing sensitive keys (BOT_TOKEN, API_ID, OWNER_IDS, etc.).
- .gitignore      : Git configuration file to prevent sensitive files (like .env or .json databases) from being uploaded to GitHub.
- requirements.txt: Python dependency list required to run the bot (aiohttp, pyrogram, mutagen, flask, python-dotenv).

4. EXECUTABLES / EXTERNAL BINARIES
----------------------------------
- mp4decrypt.exe  : External binary tool used to decrypt DRM protected MP4 audio chunks locally.
- cloudflared.exe : Cloudflare Tunnel executable used to securely expose the local Flask dashboard to the public internet.

5. MISCELLANEOUS
----------------
- info.txt        : This documentation file (Bot replies and file architecture).
- script.php / update_ui.py / refactor.py : Temporary or old utility/scratch scripts used for development or UI updates.
